# CareerPathSE Frontend

React + TailwindCSS frontend for the CareerPathSE platform.

## Setup

1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm run dev`

## Notes

- Vite dev server proxies `/api` to `http://localhost:8080`
- Auth forms are ready to use with backend endpoints:
  - `POST /api/auth/login`
  - `POST /api/auth/register`
